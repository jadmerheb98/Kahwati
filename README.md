# Kahwati App-Style Website Prototype

A multi-page, mobile-first static prototype for **Kahwati**, a luxury coffee-shop marketplace app.

## Pages

- `index.html` — ultra-clear app-style home screen and landing page
- `cafes.html` — coffee shop discovery and filters
- `menu.html` — café menu with coffee, sandwiches, snacks, desserts
- `cart.html` — delivery/takeaway checkout preview
- `rewards.html` — points and rewards wallet
- `booking.html` — book a café spot
- `account.html` — login/signup/profile preview
- `orders.html` — order history and tracking preview
- `styles.css` — full responsive design system
- `script.js` — cart, modals, tabs, fake auth interactions

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from branch**.
5. Select branch `main` and folder `/root`.
6. Open your GitHub Pages link.

## Notes

This is a static design prototype. Login, signup, cart, booking, and ordering are preview interactions only. For a real app, connect it later to a backend such as Supabase, Firebase, or a custom API.
