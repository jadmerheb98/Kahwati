# Kahwati Minimal App Preview

A mobile-first static prototype for Kahwati, inspired by smooth delivery-app UX while keeping a distinct coffee marketplace identity.

## Files
- index.html
- cafes.html
- menu.html
- cart.html
- styles.css
- script.js

## Deploy on GitHub Pages
Upload all files to your repository root, then enable GitHub Pages from Settings > Pages.

## To update your current website
Replace the full folder for best results. If you only want the visual update, replace `index.html`, `styles.css`, and `script.js`.
