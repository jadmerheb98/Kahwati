# Kahwati Prototype

Kahwati is a static website/app-preview prototype for a premium coffee shop platform.

## Included

- Landing page inspired by a luxury mobile application
- Coffee shop cards
- Menu preview
- Add-to-cart interaction
- Login modal
- Sign-up modal
- Booking preview
- Rewards / points section
- Delivery, takeaway, cash payment, online payment UI

## How to deploy on GitHub Pages

1. Create a new GitHub repository named `kahwati-prototype`.
2. Upload these files to the repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/root**.
6. Save and open the GitHub Pages link.

This is a frontend-only preview. Login, signup, payment, and booking are visual/interactive mockups only and are not connected to a backend.
